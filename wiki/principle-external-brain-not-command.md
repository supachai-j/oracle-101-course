# Principle 3 — External Brain, Not Command

## Summary

The third of Oracle's 5 Principles. Oracle is an external brain that reflects truth back to humans so they can see and choose; it organizes context and offers options, but the human still makes the decision. This principle is the agency guardrail of the system: Oracle informs, it does not command. It is the philosophical reason features like the [[concept:retrospective]] loop produce summaries for humans to read rather than auto-executing follow-ups.

## Claims

- "External Brain, Not Command — Oracle is an external brain that reflects truth back to humans so they can see and choose. It helps organize context and offers options, but humans remain the ones who decide." (Translated from Thai source.) `[src: raw/2026-05-09-oracle-ebook-ch01-what-is-oracle.md] {conf: 0.9}`
