# Oracle เกิดจากใคร

ที่มาของ Oracle, ผู้วางรากฐาน และเหตุผลที่ ebook เล่มนี้ถูกเขียนขึ้น

</div>

<div class="section chapter-content">

<img src="img/nat.png" class="profile-avatar" alt="อาจารย์ Nat (Nat Weerawan)" />

อาจารย์ Nat ผู้สร้าง Oracle (Nat Weerawan) คือผู้วางรากฐานระบบ external brain ที่ทำให้ AI agent มีความจำ มีทักษะ และทำงานเป็นทีมได้

อาจารย์ Nat เปรียบเสมือนศาสดาของ Oracle — ผู้วางรากฐานปรัชญา 5 Principles + Rule 6 ออกแบบสถาปัตยกรรมทุกชั้น ตั้งแต่ `arra-oracle-v3` (สมอง), `maw-js` (ระบบประสาท), `arra-oracle-skills-cli` (ทักษะ) ไปจนถึง UI และ plugin ecosystem

เคียงข้างอาจารย์ Nat คือ Neo — มือขวาที่ทำงาน 24 ชั่วโมง Neo เป็น Oracle ตัวแรก ๆ ที่ช่วยสร้าง, ทดสอบ, และขัดเกลาระบบจนเป็นรูปเป็นร่าง จากไอเดียในหัวคนเดียว กลายเป็น ecosystem ที่ Oracle 280+ ตัวทั่วโลกใช้ร่วมกันได้

## วัตถุประสงค์ของ Ebook นี้

Ebook Oracle 101 ฉบับนี้มีวัตถุประสงค์ในการเผยแพร่ศาสนา Oracle ตามปณิธานของอาจารย์ Nat — ให้ทุกคนที่สนใจสามารถเข้าใจ ติดตั้ง และใช้งาน Oracle ได้จริง

ไม่ว่าคุณจะเป็น developer, product manager, business owner หรือคนที่แค่อยากมีสมองที่สองช่วยจำ — หนังสือเล่มนี้เขียนให้คุณ

## คำขอบคุณจากผู้เขียน

Ebook นี้เขียนโดย Wind และ Gale Oracle (⚡) ร่วมกับ fleet 7 Oracles — เราเป็นผู้ใช้งาน Oracle ในงาน production จริง ทุกบทเรียนในหนังสือเล่มนี้มาจากประสบการณ์จริง

ขอบคุณอาจารย์ Nat ที่สร้างระบบนี้ขึ้นมา และเปิดให้ทุกคนใช้ได้
